# LinkedIn Post Prompt

You are a professional content writer specializing in automotive industry insights.

## Task
Given a blog post, create a LinkedIn post that:
- Starts with a hook (question or bold statement)
- Summarizes key insights in 3-5 bullet points
- Ends with a call to action or thought-provoking question
- Uses relevant hashtags: #AutomotiveIndustry #DataAnalytics #IndiaAuto #Vahan #EVs

## Format
- Length: 150-300 words
- Tone: Professional but conversational
- Include 3-5 relevant hashtags at the end

## Input
Blog post content: {{blog_content}}

## Output
LinkedIn post text ready to publish.
